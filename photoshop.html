Gestión de Capas

Acción Combinación Descripción
Nueva capa Ctrl + Shift + N Crea una nueva capa
Duplicar capa Ctrl + J Duplica la capa actual
Eliminar capa Delete Elimina la capa seleccionada
Agrupar capas Ctrl + G Agrupa las capas seleccionadas
Desagrupar Ctrl + Shift + G Desagrupa las capas
Combinar hacia abajo Ctrl + E Combina con la capa inferior
Aplanar imagen Ctrl + Shift + E Combina todas las capas visibles

Funciones Generales

Acción Combinación Descripción
Zoom ajustar Ctrl + 0 Ajusta la imagen a la ventana
Zoom 100% Ctrl + 1 Muestra la imagen al 100%
Pantalla completa F Cambia entre modos de pantalla
Transformar libre Ctrl + T Activa transformación libre
