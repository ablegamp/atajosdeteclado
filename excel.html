Navegación en Excel

Muévete rápidamente por las celdas y hojas de cálculo

Acción Combinación Descripción
Nueva celda Enter Confirma entrada y se mueve a la celda inferior
Celda anterior Shift + Enter Se mueve a la celda superior
Siguiente celda Tab Se mueve a la celda de la derecha
Celda izquierda Shift + Tab Se mueve a la celda de la izquierda
Inicio de fila Home Va al inicio de la fila actual
Inicio de hoja Ctrl + Home Va a la celda A1
Último dato Ctrl + End Va a la última celda con datos
Saltar a celda Ctrl + G Abre cuadro "Ir a" para saltar a una celda específica
Cambiar hoja Ctrl + Page Up/Down Cambia entre hojas del libro
Seleccionar fila Shift + Space Selecciona toda la fila actual
Seleccionar columna Ctrl + Space Selecciona toda la columna actual

Edición y Estructura

Modifica y organiza tus datos eficientemente

Acción Combinación Descripción
Editar celda F2 Edita la celda activa sin borrar contenido
Cancelar entrada Esc Cancela la entrada en la celda
Insertar fila Ctrl + Shift + + Inserta una nueva fila
Eliminar fila/columna Ctrl + - Elimina la fila o columna seleccionada
Llenar hacia abajo Ctrl + D Rellena las celdas hacia abajo
Llenar hacia derecha Ctrl + R Rellena las celdas hacia la derecha
Insertar comentario Shift + F2 Inserta un comentario en la celda
Borrar contenido Delete Borra el contenido de las celdas seleccionadas
Buscar y reemplazar Ctrl + H Abre buscar y reemplazar

Fórmulas y Funciones

Trabaja eficientemente con fórmulas y cálculos

Acción Combinación Descripción
Insertar función Shift + F3 Abre el cuadro de insertar función
Suma automática Alt + = Inserta función SUMA automáticamente
Calcular ahora F9 Calcula todas las hojas de todos los libros abiertos
Calcular hoja Shift + F9 Calcula solo la hoja activa
Mostrar fórmulas Ctrl + ` Muestra/oculta fórmulas en lugar de valores
Referencias absolutas F4 Cambia entre referencias relativas y absolutas ($)
Nombres definidos Ctrl + F3 Abre el administrador de nombres
Evaluar fórmula Alt + M + V Evalúa paso a paso una fórmula

Formato y Diseño

Mejora la apariencia de tus hojas de cálculo

Acción Combinación Descripción
Formato de celdas Ctrl + 1 Abre el cuadro de formato de celdas
Negrita Ctrl + B Aplica o quita formato de negrita
Cursiva Ctrl + I Aplica o quita formato de cursiva
Subrayado Ctrl + U Aplica o quita subrayado
Formato moneda Ctrl + Shift + $ Aplica formato de moneda
Formato porcentaje Ctrl + Shift + % Aplica formato de porcentaje
Autoajustar columna Alt + H + O + I Ajusta el ancho de columna automáticamente
Copiar formato Ctrl + Shift + C Copia el formato de la celda seleccionada
Pegar formato Ctrl + Shift + V Pega solo el formato

Herramientas profesionales para análisis de datos

Acción Combinación Descripción
Tabla dinámica Alt + N + V Inserta una tabla dinámica
Filtros automáticos Ctrl + Shift + L Activa/desactiva filtros automáticos
Ordenar datos Alt + A + S + S Abre cuadro de ordenar datos
Formato condicional Alt + H + L Aplica formato condicional
Validación de datos Alt + A + V + V Configura validación de datos
Subtotales automáticos Alt + A + B Inserta subtotales automáticos
Consolidar datos Alt + A + N Abre herramienta de consolidación
Auditoría de fórmulas Alt + M + P Rastrea precedentes de fórmula
