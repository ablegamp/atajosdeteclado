Atajos Básicos de Mac (macOS)

Los atajos esenciales para empezar con macOS: edición, navegación, Spotlight y ventanas.

Actualizado: Octubre 2025

Acción Combinación Descripción
Cortar Cmd + X Corta el elemento seleccionado
Deshacer Cmd + Z Deshace la última acción
Rehacer Cmd + Shift + Z Rehace la acción deshecha
Seleccionar todo Cmd + A Selecciona todo el contenido
Buscar Cmd + F Abre la búsqueda en documentos
Nuevo documento Cmd + N Crea un nuevo documento
Abrir documento Cmd + O Abre un documento existente
Guardar Cmd + S Guarda el documento actual
