Atajos Avanzados de Mac (macOS)

Domina Mission Control, la gestión de ventanas, Finder avanzado y utilidades clave.

Actualizado: Octubre 2025

Acción Combinación Descripción
Mission Control Ctrl + ↑ Muestra todas las ventanas y escritorios
Espacios Siguiente/Anterior Ctrl + ←/→ Cambia entre escritorios
Dock – Enfocar App Ctrl + F3 Navega por el Dock con teclado
Forzar cierre Cmd + Alt + Esc Abre diálogo de forzar cierre
Mover ventana (terceros) Ctrl + Cmd + ←/→ Depende de gestor de ventanas (Magnet/Rectangle)
