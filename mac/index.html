Atajos de Teclado en Mac (macOS): Guía Completa

Domina macOS con atajos específicos para Mac. Desde Finder hasta Mission Control y funciones avanzadas del sistema.

Actualizado: Octubre 2025

Atajos Básicos de Mac

Los shortcuts más esenciales para empezar con macOS

Ver todos los atajos básicos Ver Mac Avanzado
Acción Combinación Descripción
Copiar Cmd + C Copia el elemento seleccionado
Pegar Cmd + V Pega el contenido del portapapeles
Spotlight Cmd + Space Abre la búsqueda de Spotlight
Cambiar aplicación Cmd + Tab Cambia entre aplicaciones abiertas
Mission Control F3 Muestra todas las ventanas y espacios

Funciones del Sistema

Acción Combinación Descripción
Captura de pantalla Cmd + Shift + 3 Captura toda la pantalla
Captura de área Cmd + Shift + 4 Captura un área seleccionada
Bloquear pantalla Cmd + Ctrl + Q Bloquea la pantalla
Forzar cierre Cmd + Alt + Esc Fuerza el cierre de aplicaciones
Finder Cmd + Shift + A Abre la carpeta Aplicaciones
Utilidades Cmd + Shift + U Abre la carpeta Utilidades

Navegación del Sistema

Muévete por macOS sin usar el ratón

Acción Combinación Descripción
Dock Ctrl + F3 Muestra u oculta el Dock
Menú Apple Ctrl + F2 Activa el menú Apple
Menús de aplicación Ctrl + F8 Activa los menús de la aplicación actual
Barra de menús Ctrl + F2 Enfoca la barra de menús
Cambiar ventana Cmd + ` Cambia entre ventanas de la misma aplicación

Gestión de Ventanas

Organiza y controla las ventanas de tus aplicaciones

Acción Combinación Descripción
Minimizar ventana Cmd + M Minimiza la ventana activa
Ocultar aplicación Cmd + H Ocula la aplicación activa
Ocultar otras Cmd + Alt + H Ocula todas las ventanas excepto la activa
Mostrar escritorio F11 Muestra el escritorio
Espacios Ctrl + ↑ / Ctrl + ↓ Navega entre espacios de Mission Control
