Atajos Básicos Universales

Los shortcuts fundamentales que funcionan en casi todas las aplicaciones

Acción Combinación Descripción
Copiar Ctrl + C Copia el elemento seleccionado al portapapeles
Pegar Ctrl + V Pega el contenido del portapapeles
Cortar Ctrl + X Corta el elemento seleccionado
Deshacer Ctrl + Z Deshace la última acción
Rehacer Ctrl + Y Rehace la acción deshecha
Seleccionar todo Ctrl + A Selecciona todo el contenido
Buscar Ctrl + F Abre el cuadro de búsqueda
Guardar Ctrl + S Guarda el archivo actual
Imprimir Ctrl + P Abre el cuadro de diálogo de impresión
Nuevo Ctrl + N Crea un nuevo archivo o ventana

Navegación del Sistema

Muévete por Windows sin usar el ratón

Acción Combinación Descripción
Menú Inicio Win Abre o cierra el menú Inicio
Explorador de archivos Win + E Abre el Explorador de archivos
Escritorio Win + D Muestra u oculta el escritorio
Configuración Win + I Abre la aplicación Configuración
Buscar Win + S Abre la búsqueda de Windows
Cambiar aplicación Alt + Tab Cambia entre aplicaciones abiertas
Vista de tareas Win + Tab Abre la vista de tareas y escritorios virtuales
Panel de control Win + R Abre el cuadro de diálogo Ejecutar

Gestión de Ventanas

Organiza y controla las ventanas de tus aplicaciones

Acción Combinación Descripción
Cerrar ventana Alt + F4 Cierra la ventana activa
Minimizar ventana Win + ↓ Minimiza la ventana actual
Maximizar ventana Win + ↑ Maximiza la ventana actual
Ajustar izquierda Win + ← Ajusta la ventana a la mitad izquierda
Ajustar derecha Win + → Ajusta la ventana a la mitad derecha
Minimizar todas Win + M Minimiza todas las ventanas
Restaurar ventanas Win + Shift + M Restaura las ventanas minimizadas

Centro de actividades Win + A Abre el centro de actividades
Captura de pantalla Win + Shift + S Abre la herramienta de recorte
Símbolos especiales Win + ; Abre el panel de emoji y símbolos
Monitor de recursos Win + Pause Abre información del sistema
