Atajos de Microsoft Edge

Pestañas, colecciones y pestañas verticales con atajos de teclado.

Actualizado: Octubre 2025

En esta página: Pestañas Historial y Descargas Herramientas

Pestañas

Acción Combinación Descripción
Nueva pestaña Ctrl + T / Cmd + T Abrir nueva pestaña
Nueva ventana Ctrl + N / Cmd + N Abrir nueva ventana de Edge
Abrir Colecciones Ctrl + Shift + Y Mostrar panel de Colecciones
Pestañas verticales Ctrl + Shift + , Alternar vista de pestañas verticales
Cambiar entre pestañas Ctrl + Tab / Ctrl + Shift + Tab Navegar por pestañas
Ir a pestaña (1–8) Ctrl + 1..8 / Cmd + 1..8 Salta a una pestaña por posición
Última pestaña Ctrl + 9 / Cmd + 9 Ir a la última pestaña abierta

Historial y descargas

Acción Combinación Descripción
Historial Ctrl + H / Cmd + Y Abrir el historial de navegación
Descargas Ctrl + J / Cmd + J Abrir la página de descargas
Atrás / Adelante Alt + ←/→ / Cmd + [/] Navegar por el historial de la pestaña
Borrar datos de navegación Ctrl + Shift + Delete / Cmd + Shift + Delete Abrir diálogo de borrado

Herramientas y opciones

Acción Combinación Descripción
Buscar en página Ctrl + F / Cmd + F Abrir búsqueda en la página
DevTools Ctrl + Shift + I / Cmd + Opt + I Abrir herramientas de desarrollador
Barra de direcciones Ctrl + L / Cmd + L Foco en la barra de direcciones
Ventana InPrivate Ctrl + Shift + N / Cmd + Shift + N Abrir una ventana InPrivate
Zoom +/- / reset Ctrl + +/−/0 / Cmd + +/−/0 Acercar, alejar o restablecer zoom
Cerrar pestaña / ventana Ctrl + W / Ctrl + Shift + W — Cmd + W / Cmd + Shift + W Cerrar pestaña actual o ventana
