Navega a Toda Velocidad: Los Mejores Atajos de Teclado para Google Chrome

28 de Octubre, 2025 8 min de lectura

Gestión de Pestañas y Ventanas

Controla tus pestañas y ventanas sin esfuerzo.

Acción Combinación Descripción
Abrir nueva pestaña Ctrl + T Inicia una nueva pestaña en blanco al instante.
Cerrar pestaña actual Ctrl + W Cierra rápidamente la pestaña en la que estás.
Reabrir pestaña cerrada Ctrl + Mayús + T Recupera la última pestaña que cerraste.
Nueva ventana Ctrl + N Abre una nueva ventana de Chrome.
Nueva ventana de incógnito Ctrl + Mayús + N Abre una ventana en modo incógnito para navegar de forma privada.
Ir a pestaña específica Ctrl + 1-8 Salta a la pestaña en la posición indicada.
Ir a la última pestaña Ctrl + 9 Salta a la última pestaña abierta.

Navegación Rápida

Muévete por la web a la velocidad de la luz.

Acción Combinación Descripción
Página anterior Alt + ← Vuelve a la página anterior en tu historial de navegación.
Página siguiente Alt + → Avanza a la página siguiente en tu historial.
Recargar página F5 o Ctrl + R Vuelve a cargar la página actual.
Seleccionar barra de direcciones Ctrl + L o Alt + D Selecciona la URL para escribir una nueva dirección o buscar.
Buscar en la página Ctrl + F Abre la barra de búsqueda para encontrar texto en la página.

Herramientas y Funciones

Accede a las herramientas integradas de Chrome al instante.

Acción Combinación Descripción
Historial Ctrl + H Abre la página de historial en una nueva pestaña.
Descargas Ctrl + J Muestra la página de descargas.
Guardar marcador Ctrl + D Guarda la página actual en tus marcadores.
Mostrar/ocultar barra de marcadores Ctrl + Mayús + B Alterna la visibilidad de la barra de marcadores.
Administrador de tareas de Chrome Mayús + Esc Abre el administrador para ver el consumo de recursos.
Herramientas de desarrollo Ctrl + Mayús + I Abre las herramientas para desarrolladores web.
Borrar datos de navegación Ctrl + Mayús + Supr Abre el menú para borrar historial, cookies y caché.
