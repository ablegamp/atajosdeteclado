Preguntas frecuentes

¿Cómo abrir una nueva pestaña en Chrome?
Usa Ctrl + T (Windows) o Cmd + T (Mac).
¿Cómo reabrir la última pestaña cerrada?
Ctrl + Shift + T (Windows) o Cmd + Shift + T (Mac).
¿Cómo ver descargas rápidamente?
Ctrl + J (Windows) o Cmd + J (Mac).

Historial y descargas

Acción Combinación Descripción
Nueva pestaña Ctrl + T / Cmd + T Abrir nueva pestaña
Reabrir pestaña cerrada Ctrl + Shift + T / Cmd + Shift + T Recuperar la última pestaña cerrada
Cambiar entre pestañas Ctrl + Tab / Ctrl + Shift + Tab Navegar por pestañas
Ir a pestaña (1–8) Ctrl + 1..8 / Cmd + 1..8 Salta directamente a la pestaña por posición
Última pestaña Ctrl + 9 / Cmd + 9 Ir a la última pestaña abierta
