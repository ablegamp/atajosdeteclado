Macros y VBA

Automatiza tareas repetitivas con macros y programación VBA

Acción Combinación Descripción
Grabar macro Alt + T + M + R Inicia la grabación de una nueva macro
Detener grabación Alt + T + M + M Detiene la grabación de la macro
Editor VBA Alt + F11 Abre el editor de Visual Basic
Ejecutar macro Alt + F8 Abre el cuadro de diálogo de macros
Referencias VBA Alt + T + O + O Abre referencias de VBA

Análisis de Datos Avanzado

Herramientas profesionales para análisis complejo

Acción Combinación Descripción
Power Query Alt + A + P + Q Abre el editor de Power Query
Análisis rápido Ctrl + Q Abre análisis rápido de datos
Gráfico recomendado Alt + F1 Crea gráfico recomendado automáticamente
Herramientas de datos Alt + A + Y + 2 Abre herramientas avanzadas de datos
