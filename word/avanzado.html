Word Avanzado

Atajos profesionales: estilos, navegación por encabezados, revisión y control de cambios.

Estilos y Formato Avanzado

Acción Combinación Descripción
Aplicar estilo Ctrl + Shift + S Abre el cuadro de Estilos
Aumentar/disminuir nivel de lista Tab / Shift + Tab Cambia el nivel de sangría en listas
Mayúsculas/minúsculas Shift + F3 Alterna entre mayúsculas, minúsculas y título

Navegación por Encabezados

Acción Combinación Descripción
Panel de navegación Ctrl + F Abre el panel para ir por encabezados/páginas
Insertar salto de página Ctrl + Enter Inicia una nueva página
Marcadores Ctrl + Shift + F5 Administrar marcadores

Revisión y Control de Cambios

Acción Combinación Descripción
Insertar comentario Ctrl + Alt + M Añade un comentario
Control de cambios Ctrl + Shift + E Activa o desactiva el control de cambios
Aceptar cambio Alt + Shift + A Acepta el cambio actual
