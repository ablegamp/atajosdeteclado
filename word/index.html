Navegación en Word

Muévete rápido por documentos y paneles

Acción Combinación Descripción
Mover palabra a palabra Ctrl + →/← Salta al inicio de la palabra siguiente/anterior
Inicio/fin de línea Home / End Ir al inicio/fin de la línea actual
Ir al inicio/fin del documento Ctrl + Home/End Salta al principio o final del documento
Zoom ± Ctrl + Rueda Aumenta o reduce el zoom

Edición y Portapapeles

Escribir, seleccionar y gestionar contenido

Acción Combinación Descripción
Copiar / Cortar / Pegar Ctrl + C / X / V Gestiona el portapapeles rápidamente
Deshacer / Rehacer Ctrl + Z / Y Revertir o repetir acciones
Seleccionar palabra Ctrl + Shift + →/← Extiende la selección por palabras
Buscar y reemplazar Ctrl + F / H Encuentra texto y reemplázalo

Formato y Diseño

Aplica formato al texto y a los párrafos

Acción Combinación Descripción
Negrita / Cursiva / Subrayado Ctrl + B / I / U Formato directo de caracteres
Alinear texto Ctrl + L/E/R/J Alineación izquierda, centrado, derecha, justificado
Tamaño de fuente ± Ctrl + ] / [ Aumenta o reduce el tamaño de la fuente
Estilos rápidos Ctrl + Shift + S Aplica estilos guardados

Revisión y Comentarios

Colabora y corrige más rápido

Acción Combinación Descripción
Insertar comentario Ctrl + Alt + M Añade un comentario en la posición del cursor
Ortografía y gramática F7 Abre el revisor ortográfico
Cambios controlados Ctrl + Shift + E Activa/desactiva control de cambios
Aceptar cambio Alt + Shift + A Acepta el cambio actual
