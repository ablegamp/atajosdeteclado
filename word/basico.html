Atajos de teclado en Word Básico

Atajos de teclado esenciales para empezar: navegación, selección y formato simple.

Navegación y Selección

Muévete y selecciona texto con precisión

Acción Combinación Descripción
Mover palabra Ctrl + →/← Salta por palabras
Seleccionar palabra Ctrl + Shift + →/← Selecciona palabra a palabra
Inicio/fin de línea Home / End Ir al inicio/fin de línea
Inicio/fin de documento Ctrl + Home/End Salta al principio o al final

Portapapeles y Edición

Acción Combinación Descripción
Copiar / Cortar / Pegar Ctrl + C / X / V Gestiona el portapapeles
Deshacer / Rehacer Ctrl + Z / Y Revierte o repite acciones
Seleccionar todo Ctrl + A Selecciona todo el documento
Buscar y Reemplazar Ctrl + F / H Encuentra o reemplaza texto

Formato básico

Acción Combinación Descripción
Negrita / Cursiva / Subrayado Ctrl + B / I / U Formato directo
Aumentar/Disminuir tamaño Ctrl + ] / [ Ajusta el tamaño de fuente
Alinear texto Ctrl + L/E/R/J Izquierda, centrado, derecha, justificado
