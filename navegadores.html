Atajos de Navegadores

Navega más rápido por la web. Atajos para Chrome, Edge, Firefox y herramientas de desarrollo.

Google Chrome

Acción Combinación Descripción
Nueva pestaña Ctrl + T Abre una nueva pestaña
Cerrar pestaña Ctrl + W Cierra la pestaña actual
Reabrir pestaña Ctrl + Shift + T Reabre la última pestaña cerrada
Actualizar página F5 Actualiza la página actual
Herramientas desarrollador F12 Abre las herramientas de desarrollador
Historial Ctrl + H Abre el historial de navegación
Descargas Ctrl + J Abre la página de descargas
Modo incógnito Ctrl + Shift + N Abre ventana de navegación privada

Microsoft Edge

Acción Combinación Descripción
Colecciones Ctrl + Shift + Y Abre el panel de Colecciones
Búsqueda vertical Ctrl + Shift + A Abre búsqueda en la barra lateral
Modo inmersivo F11 Activa/desactiva modo de pantalla completa
Favoritos Ctrl + Shift + O Abre el administrador de favoritos

Mozilla Firefox

Acción Combinación Descripción
Marcadores Ctrl + Shift + B Muestra/oculta la barra de marcadores
Modo privado Ctrl + Shift + P Abre ventana de navegación privada
Complementos Ctrl + Shift + A Abre el administrador de complementos
Consola web Ctrl + Shift + K Abre la consola web para desarrolladores
