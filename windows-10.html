Funciones Específicas de Windows 10

Atajos únicos y funciones exclusivas de Windows 10

Acción Combinación Descripción
Centro de actividades Win + A Abre el centro de actividades (notificaciones)
Cortana Win + C Activa Cortana (si está habilitada)
Escritorios virtuales Win + Tab Abre la vista de tareas y escritorios virtuales
Crear escritorio virtual Win + Ctrl + D Crea un nuevo escritorio virtual
Cerrar escritorio virtual Win + Ctrl + F4 Cierra el escritorio virtual actual
Cambiar escritorio Win + Ctrl + ←/→ Cambia entre escritorios virtuales
Game Bar Win + G Abre Xbox Game Bar para grabar
Proyectar Win + P Abre opciones de proyección de pantalla
Conectar dispositivos Win + K Conectar dispositivos inalámbricos
Recorte y anotación Win + Shift + S Abre herramienta de recorte y anotación

Ajuste de Ventanas (Snap)

Funcionalidad mejorada de Snap en Windows 10

Acción Combinación Descripción
Snap izquierda Win + ← Ajusta ventana a la mitad izquierda
Snap derecha Win + → Ajusta ventana a la mitad derecha
Cuadrante superior izquierdo Win + ← + ↑ Ajusta a cuadrante superior izquierdo
Cuadrante inferior izquierdo Win + ← + ↓ Ajusta a cuadrante inferior izquierdo
Cuadrante superior derecho Win + → + ↑ Ajusta a cuadrante superior derecho
Cuadrante inferior derecho Win + → + ↓ Ajusta a cuadrante inferior derecho

Explorador de Archivos

Navegación mejorada en el Explorador de Windows 10

Acción Combinación Descripción
Acceso rápido Ctrl + 1 Va al panel de Acceso rápido
OneDrive Ctrl + 2 Va a OneDrive (si está configurado)
Este equipo Ctrl + 3 Va a "Este equipo"
Red Ctrl + 4 Va a ubicaciones de red
Vista previa Alt + P Activa/desactiva panel de vista previa
Detalles Alt + Shift + P Activa/desactiva panel de detalles
