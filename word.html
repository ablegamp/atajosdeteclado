Formato de Texto

Formatea tu texto rápidamente sin usar el ratón

Acción Combinación Descripción
Negrita Ctrl + B Aplica o quita formato de negrita
Cursiva Ctrl + I Aplica o quita formato de cursiva
Subrayado Ctrl + U Aplica o quita subrayado
Centrar texto Ctrl + E Centra el párrafo
Justificar texto Ctrl + J Justifica el párrafo
Alinear izquierda Ctrl + L Alinea el párrafo a la izquierda
Alinear derecha Ctrl + R Alinea el párrafo a la derecha
Aumentar fuente Ctrl + Shift + > Aumenta el tamaño de la fuente
Disminuir fuente Ctrl + Shift + < Disminuye el tamaño de la fuente
Copiar formato Ctrl + Shift + C Copia el formato del texto seleccionado
Pegar formato Ctrl + Shift + V Pega el formato copiado

Navegación por el Documento

Muévete eficientemente por documentos largos

Acción Combinación Descripción
Inicio de documento Ctrl + Home Va al inicio del documento
Final de documento Ctrl + End Va al final del documento
Palabra siguiente Ctrl + → Se mueve a la siguiente palabra
Palabra anterior Ctrl + ← Se mueve a la palabra anterior
Párrafo siguiente Ctrl + ↓ Se mueve al siguiente párrafo
Párrafo anterior Ctrl + ↑ Se mueve al párrafo anterior
Ir a página Ctrl + G Abre cuadro para ir a página específica
Página arriba Page Up Se desplaza una página hacia arriba
Página abajo Page Down Se desplaza una página hacia abajo

Edición Avanzada

Herramientas para edición eficiente de texto

Acción Combinación Descripción
Seleccionar palabra Ctrl + Shift + → Selecciona la palabra siguiente
Seleccionar párrafo Ctrl + Shift + ↓ Selecciona hasta el final del párrafo
Insertar hipervínculo Ctrl + K Inserta un hipervínculo
Salto de página Ctrl + Enter Inserta un salto de página
Salto de línea Shift + Enter Inserta salto de línea sin nuevo párrafo
Borrar palabra izquierda Ctrl + Backspace Borra la palabra a la izquierda del cursor
Borrar palabra derecha Ctrl + Delete Borra la palabra a la derecha del cursor
Mayúsculas/minúsculas Shift + F3 Cambia entre mayúsculas y minúsculas
Insertar fecha Alt + Shift + D Inserta la fecha actual

Revisión y Colaboración

Herramientas para revisar y colaborar en documentos

Acción Combinación Descripción
Ortografía y gramática F7 Inicia revisión ortográfica y gramatical
Sinónimos Shift + F7 Busca sinónimos de la palabra seleccionada
Contar palabras Ctrl + Shift + G Abre el contador de palabras
Insertar comentario Ctrl + Alt + M Inserta un comentario
Control de cambios Ctrl + Shift + E Activa/desactiva control de cambios
Vista preliminar Ctrl + F2 Muestra vista preliminar del documento
