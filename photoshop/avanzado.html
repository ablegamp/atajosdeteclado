Atajos de teclado en Photoshop Avanzado

Atajos de teclado profesionales: máscaras avanzadas, ajustes precisos y técnicas expertas.

Máscaras Avanzadas

Acción Combinación Descripción
Máscara de capa Alt + Click en máscara Activa/desactiva la máscara de capa
Nueva máscara Alt + Ctrl + N Crea una nueva máscara de capa
Eliminar máscara Alt + Ctrl + R Elimina la máscara de capa
Máscara vectorial Alt + Ctrl + V Crea una máscara vectorial

Ajustes Avanzados

Acción Combinación Descripción
Curvas Ctrl + M Abre el ajuste de curvas
Niveles Ctrl + L Abre el ajuste de niveles
Tono/Saturación Ctrl + U Abre el ajuste de tono/saturación
Equilibrio de color Ctrl + B Abre el ajuste de equilibrio de color
Filtro de cámara raw Shift + Ctrl + A Abre el filtro Camera Raw

Técnicas Expertas

Acción Combinación Descripción
Libre transformación Ctrl + T Activa el modo de transformación libre
Deformar Ctrl + Shift + T Abre la herramienta de deformación
Mezcla avanzada Alt + Doble click en capa Abre opciones de mezcla avanzada
Filtro licuar Shift + Ctrl + X Abre el filtro Licuar
Desenfocar superficie Shift + Ctrl + C Aplica desenfoque de superficie
